# Installation

## From PyPI

```bash
pip install pvliesdonk-reqeng-mcp
```

## From Docker

```bash
docker pull ghcr.io/pvliesdonk/reqeng-mcp:latest
```

## From source

```bash
git clone https://github.com/pvliesdonk/reqeng-mcp
cd reqeng-mcp
uv sync --all-extras --all-groups
```
